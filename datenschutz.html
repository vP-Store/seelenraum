2. Überblick über die Datenverarbeitung

Seelenraum ist eine spirituelle Praxis-App als Progressive Web App (PWA). Die App speichert deine persönlichen Daten ausschließlich lokal in deinem Browser (localStorage). Es findet keine automatische Übermittlung deiner Inhalte an einen Server statt.

Diese Datenschutzerklärung beschreibt, welche Daten beim Besuch der Website und bei der Nutzung der App verarbeitet werden.

3. Hosting (GitHub Pages)

Diese Website wird über GitHub Pages gehostet. GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA, stellt die Infrastruktur bereit. Beim Aufruf der Seite erhebt GitHub automatisch technische Zugriffsdaten (IP-Adresse, Browsertyp, Betriebssystem, aufgerufene Seiten, Uhrzeit). Dies ist für den Betrieb der Website technisch erforderlich.

GitHub verarbeitet diese Daten auf Servern in den USA. GitHub ist nach dem EU-US Data Privacy Framework zertifiziert. Weitere Informationen: GitHub Privacy Statement.

Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einem sicheren und effizienten Betrieb der Website).

4. Lokale Speicherung (localStorage)

Die Seelenraum-App speichert folgende Daten ausschließlich lokal in deinem Browser:

  • Tagebuch-Einträge
  • Stimmungs-Check-ins und Verlauf
  • Praxis-Serie (Streak) und Meilensteine
  • Wochen-Intention
  • Favoriten
  • Name und Geburtsdatum (für Personalisierung)
  • App-Einstellungen (Hell/Dunkel-Modus)

Diese Daten werden nicht an uns oder Dritte übermittelt. Du kannst jederzeit in den App-Einstellungen unter „Alles zurücksetzen" alle lokal gespeicherten Daten löschen. Ein Export deiner Tagebuch-Einträge ist über die Einstellungen als .txt-Datei möglich.

Rechtsgrundlage: Die lokale Speicherung erfolgt auf deinem Endgerät und unterliegt nicht der DSGVO, da keine Datenverarbeitung durch uns stattfindet. Der Zugriff auf den localStorage dient der Bereitstellung der App-Funktionen und erfolgt auf Grundlage von Art. 6 Abs. 1 lit. b DSGVO (Vertragserfüllung).

5. Service Worker (Offline-Funktion)

Die App nutzt einen Service Worker, um die App offline verfügbar zu machen. Der Service Worker speichert eine Kopie der App-Dateien (HTML, CSS, JavaScript, Icons) im Browser-Cache. Es werden keine personenbezogenen Daten durch den Service Worker verarbeitet.
